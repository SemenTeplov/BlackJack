# BlackJack
Exam project Blackjack

Rules Blackjack

Vin:

Gather 21 points or gather more points, than the dealer.

Points:

- Cards: 6, 7, 8, 9, 10 match 6, 7, 8, 9, 10 points.

- Cards: Knight, Queen, King match 10 points.

- Card: Ace match 11 or 1.
